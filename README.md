# Deskripsi
Repository ini berisi **Proyek Pemrograman Dasar**, yaitu Aplikasi **Manajemen Inventaris Sewa**. aplikasi ini dirancang dengan mengimplementasikan struktur data struct untuk merepresentasikan objek item dengan atribut kompleks seperti ID unik, nama item, jenis kategori, harga sewa, dan status ketersediaan. Desain UI aplikasi Manajemen Inventaris Barang Sewaan dibuat dengan menggunakan Command Line Interface (CLI), yang dimana Interaksi antara pengguna dan sistem terjadi melalui terminal/konsol dengan menggunakan I/O standar sehingga lebih sederhana dan lebih mudah diakses dengan tampilan yang sederhana tetapi tetap jelas dan memenuhi kategori yang ditentukan oleh modul. Seluruh kode ditulis menggunakan **bahasa C++** sesuai dengan ketentuan pada modul tugas.
